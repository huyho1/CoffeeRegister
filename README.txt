This program prompts the user to enter either 1, 2 or 3 and it will calculate
depending on user input, either the monthly sales of an item, the overall sales
for all items, or find the most popular item. 

To compile the program, run the command:
make

To run the program, run the command:
./register

and then enter either 1, 2 or 3 for the kind of program that you'd like to run.
